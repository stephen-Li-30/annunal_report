# 常见缺失数据补充方法

当某字段在前序来源中未查询到有效数据时，才进入下一来源级别继续补充。本文件的所有方法都必须服从统一规则：**按第1级 → 第2级 → 第3级 → 第4级 → 第5级顺序查询，哪一级先查询到有效数据，就直接使用哪一级数据，不再按字段特性重新选择来源。**

## ROE数据补充

**方法一：季度ROE查找**
```
搜索关键词："{公司名} ROE 净资产收益率 {年份}Q3/Q4"
数据来源：今日头条、雪球、新浪财经
```

**方法二：年度ROE查找**
```
搜索关键词："{公司名} ROE 股东权益回报率 {年份}"
数据来源：雪球、理杏仁、东方财富
```

**方法三：ROE反推法（精确值）**
```
公式：ROE = 净利润 / 净资产
适用条件：已知净利润和净资产
```

**ROE数据差异处理**：Q3 ROE通常高于年度ROE，必须同时报告季度ROE和年度ROE，标注数据来源和时间点。

## PE/PB数据补充

**方法一：专业数据平台查找**
```
搜索关键词："{公司名} {股票代码} PE PB 市盈率 市净率"
数据来源：雪球、理杏仁、东方财富
```

**方法二：多市场数据收集**
注意：不同市场PE/PB可能不同（汇率、流动性差异），需分别标注。

**方法三：历史百分位估算**
```
历史PE区间 → 当前PE → 历史百分位
注：精确历史百分位需Wind/Bloomberg专业数据库
```

## 资产负债表数据补充

**方法一：直接查找法**
```
搜索关键词："{公司名} 资产负债表 总资产 总负债 净资产 {年份}"
数据源：新浪财经资产负债表、雪球财报解析
```

**方法二：财务比率推算法**
```
已知：总负债、资产负债比
推算：总资产 = 总负债 x 资产负债比
      净资产 = 总资产 - 总负债
标注：计算值（基于资产负债比推算）
```

**方法三：ROE反推净资产法**
```
已知：净利润、ROE
推算：净资产 = 净利润 / ROE
标注：计算值（基于ROE反推）
```

**验证公式**：总资产 = 总负债 + 净资产（误差<10%为合理）

## 数据补充执行顺序

| 数据类型 | 统一执行规则 | 备注 |
|---------|------------|------|
| 全部字段 | 必须按五级顺序逐级查询 | 哪一级先查询到有效数据，就直接使用哪一级 |
| 若当前级别无有效数据 | 才允许进入下一级 | 模糊线索不算有效数据 |
| 若五级均无结果 | 标注数据缺口 | 需说明原因 |

## 数据补充验证清单

| 验证项 | 验证方法 | 通过标准 |
|--------|---------|---------|
| ROE合理性 | 季度ROE vs 年度ROE | 差异可解释 |
| PE/PB一致性 | 美股 vs 港股 | 差异合理 |
| 资产负债平衡 | 总资产 = 总负债 + 净资产 | 误差<10% |
| 数据来源标注 | 每项数据有来源 | 100%标注 |
| 计算值说明 | 说明推算逻辑 | 逻辑清晰可追溯 |

## 案例：阿里巴巴 FY2025

1. ROE：Q3 18.7%（今日头条 2025-03-14），年度 8.9%（雪球 2024-12-28）
2. PE/PB：美股PE 16.5 PB 1.47，港股PE 22.54 PB 2.02
3. 总资产：约2014亿美元（资产负债比2.00推算），总负债1007亿，净资产928亿
4. 验证：1007 + 928 = 1935亿 ≈ 2014亿（误差4%）
